# Secret backend: SOPS + age (not Vault)

## Rationale

Solo operator, no HA Vault possible without 3rd node. Single-node Vault = SPOF blocking all secret-dependent workloads on failure. SOPS+age has zero operational overhead, works identically on both planes (Tofu SOPS provider + ArgoCD SOPS plugin / ESO SOPS backend). Stepping stone: migrate to Vault if 3rd node added or dynamic secrets needed. CRITICAL: age private key must be backed up to offline storage.

## Affected Files

- `providers.tf`
- `variables.tf`
