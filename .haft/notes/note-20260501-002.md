# Networking: VLAN-aware bridges + SDN NAT zones per node

## Rationale

Both nodes: vmbr0 VLAN-aware, VLANs 3 (mgmt/DHCP) and 4 (secondary/DHCP). node3 adds vmbr1 (10G SFP+, MTU 9000, sniff/mirror) and vmbr2 (2.5GbE, VLANs 2-4). SDN internal zones: node3=10.13.0.0/24 SNAT via 10.0.3.13, node4=10.14.0.0/24 SNAT via 10.0.3.14. DNS: dnsmasq static leases only, domain <node>.sf.cktom.cyou.
